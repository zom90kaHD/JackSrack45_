# Assets Folder

## Instruksi Menyimpan Gambar Avatar

1. **Simpan gambar karakter dengan nama: `avatar.png`** ke folder ini
2. Ukuran rekomendasi: 300x400 pixel (PNG format)
3. Setelah disimpan, gambar akan otomatis tampil di header halaman

Gambar akan ditampilkan dengan:
- Ukuran: 150x200 pixel
- Animasi bounce (naik-turun)
- Shadow effect untuk kedalaman
- Positioning: Sebelah kiri dari judul "Link JackSrack45"

Pastikan file bernama persis **`avatar.png`** agar Website dapat membaca gambar dengan benar.
